# PyMISR

This repository aims to implement and replicate the algorithm and results presented in the paper **"Discovering nuclear models from symbolic machine learning"**. 

It is based on and inspired by the original implementation available at the following GitHub repository:
[munozariasjm/nuclear-misr](https://github.com/munozariasjm/nuclear-misr)

## Overview

The primary goal of this project is to use symbolic regression and machine learning techniques to discover interpretable models for physics, in this case specifically focusing on nuclear binding energy. This involves recreating the methodology described in the original paper to reproduce its findings and potentially extend them.

## Directory Structure

* `src/`: Source code including the implementation of the binding energy models.
* `datasets/`: Data used for training and evaluating the models.
* `notebooks/`: Jupyter notebooks for exploration and analysis.
* `outputs/` & `results/`: Directories for storing generated models, figures, and execution results.
